Mining Anomalies Using Traffic Feature Distributions

Course Project

  • An implementation of the entropy prediction algorithm presented by Lakhina in 2005, which is an anomaly detection method based on PCA traffic model.
  • It aggregates the traffic from the individual source IP addresses, it predicts the entropies of destination IP addresses, destination ports and source ports over the set of context NetFlows for each source. An anomaly is determined as the normalized sum of residual entropy over all three dimensions of the context space.

SDN-based ARP spoofing detection application

Course Project

  • Due to the limitations of traditional network detection methods for ARP spoofing, our project has developed an ONOS application using SDN technology to detect ARP attacks.
  • The controller first caches IP-MAC address pairs via DHCP traffic. The controller monitors the network by controlling the forwarding rules of the switch. When illegal ARP packets are detected, it logs the information and issues rules to the controller to prohibit the attacker from accessing the network.
